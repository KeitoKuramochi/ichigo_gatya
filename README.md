# ichigo_gatya

ICHIGO(Optimism上のERC-20トークン)を使った、実機ガチャガチャの決済システム。

QRコードから決済ページを開き、各自のウォレットでICHIGOを送金すると、段ボール製の実機ガチャガチャのロックが解除される仕組みです。

## 構成

- `payment/` — 決済ページ(Vercelで公開)。ウォレット接続 → 残高確認 → 送金のみのシンプルなページ
- `bridge/` — 送金をOptimism上でオンチェーン検証し、ESP32(マイコン)に信号を送るローカルサーバー(Node.js)
- `firmware/` — ESP32用のArduinoスケッチ。WiFi経由でサーボモーターを制御し、実機のロックを解除する
